### ✅ Что работает:
- **LoRA fine-tuning** — потребляет **намного меньше памяти**, чем полное обучение
- **Модель 1.7B** на LoRA способна влезть в 12 ГБ памяти RTX 3060 с:
  - Batch size = 4 (вместо 16-32 на RTX 4090)
  - Gradient accumulation
  - Flash Attention (оптимизирует использование памяти)

### ⚠️ Критические проблемы:

1. **Недостаточно видео-памяти для некоторых операций:**
   - Инференс с полной моделью может быть граничным (требует ~8 ГБ)
   - Evaluation на больших батчах может вызвать OOM

2. **Скорость обучения:**
   - Ожидайте **3-5x замедление** по сравнению с RTX 4090
   - Каждая эпоха займёт заметно больше времени

3. **Обязательные требования:**
   ```bash
   pip install -U flash-attn --no-build-isolation  # Критично для экономии памяти!
   ```

4. **Рекомендованные настройки под RTX 3060:**
   ```bash
   BATCH_SIZE=2              # Вместо рекомендованного 4
   GRADIENT_ACCUMULATION=2   # Увеличить шаги аккумуляции
   LR=2e-6                   # Стандартно
   EPOCHS=10                 # Максимум
   LORA_RANK=8               # Вместо 16 (можно пробовать)
   ```

### 🔧 Как оптимизировать:

1. **Убедитесь в установке Flash Attention** — это даст наибольший выигрыш по памяти
2. **Используйте меньший batch size** (2 вместо 4)
3. **Подготовьте малый датасет** для testing (недостаток памяти)
4. **Отключите сохранение checkpoint'ов** для каждой эпохи, если нужно место

## 🔧 Критические параметры для RTX 3060:

| Параметр | Рекомендация | Причина |
|----------|--------------|---------|
| **Batch size** | 2 (не 4) | 12GB VRAM |
| **Gradient accumulation** | 2 | Имитирует batch 4 без памяти |
| **LoRA rank** | 8-16 | 16 — стандарт, 8 если OOM |
| **Learning rate** | 2e-6 | VITS чувствителен, не больше |
| **Epochs** | 5-10 | Переобучение после 10 |
| **Audio sample rate** | 24 kHz | Критично для VITS |

## ⚠️ Специфика восточноармянского:

1. **Фонемы**: некоторые звуки отличаются от западноармянского
   - Убедитесь, что фонетическая транскрипция соответствует **HYE**, не HYW
   - Может потребоваться адаптация фонемного инвентаря

   - Минимум: **50-100 высококачественных примеров** (даже для LoRA)
   - Оптимально: **1000+ примеров** для хорошего качества

3. **Синтез голоса**:
   - LoRA адаптирует только стиль/акцент, не меняет базовый голос
   - Если вам нужен другой голос — нужен другой базовый checkpoint

### 5. **utils/logger.py**

```python
import logging
import os
from datetime import datetime

def get_logger(name: str, log_dir: str = "outputs/logs"):
    """Create a logger with file and console handlers."""
    os.makedirs(log_dir, exist_ok=True)
    
    logger = logging.getLogger(name)
    logger.setLevel(logging.DEBUG)
    
    # File handler
    timestamp = datetime.now().strftime("%Y%m%d_%H%M%S")
    fh = logging.FileHandler(f"{log_dir}/train_{timestamp}.log")
    fh.setLevel(logging.DEBUG)
    
    # Console handler
    ch = logging.StreamHandler()
    ch.setLevel(logging.INFO)
    
    # Formatter
    formatter = logging.Formatter(
        '%(asctime)s - %(name)s - %(levelname)s - %(message)s'
    )
    fh.setFormatter(formatter)
    ch.setFormatter(formatter)
    
    logger.addHandler(fh)
    logger.addHandler(ch)
    
    return logger
```

### 6. **utils/audio.py**

```python
import librosa
import numpy as np
import torch

def load_audio(path: str, sr: int = 16000) -> np.ndarray:
    """Load audio file and resample to target sample rate."""
    audio, current_sr = librosa.load(path, sr=None)
    if current_sr != sr:
        audio = librosa.resample(audio, orig_sr=current_sr, target_sr=sr)
    return audio

def resample_audio(audio: np.ndarray, orig_sr: int, target_sr: int) -> np.ndarray:
    """Resample audio to target sample rate."""
    if orig_sr == target_sr:
        return audio
    return librosa.resample(audio, orig_sr=orig_sr, target_sr=target_sr)
```

```python
import os
import csv
import random
from pathlib import Path

# Параметры
DATA_RAW = Path("data_raw")
METADATA_FILE = DATA_RAW / "metadata.csv"
TRAIN_RATIO = 0.8
VAL_RATIO = 0.1
TEST_RATIO = 0.1
RANDOM_SEED = 42

random.seed(RANDOM_SEED)

# Создайте структуру папок
for split in ["train", "val", "test"]:
    (DATA_RAW / split / "audio").mkdir(parents=True, exist_ok=True)

# Прочитайте metadata.csv
with open(METADATA_FILE, 'r', encoding='utf-8') as f:
    rows = list(csv.reader(f, delimiter='|'))

# Перемешайте и разделите
random.shuffle(rows)
train_size = int(len(rows) * TRAIN_RATIO)
val_size = int(len(rows) * VAL_RATIO)

train_rows = rows[:train_size]
val_rows = rows[train_size:train_size + val_size]
test_rows = rows[train_size + val_size:]

# Создайте metadata.csv для каждого split
splits = {
    'train': train_rows,
    'val': val_rows,
    'test': test_rows
}

for split, split_rows in splits.items():
    output_csv = DATA_RAW / split / "metadata.csv"
    with open(output_csv, 'w', encoding='utf-8', newline='') as f:
        writer = csv.writer(f, delimiter='|')
        writer.writerows(split_rows)
    print(f"{split}: {len(split_rows)} файлов")

print("✅ Данные разделены успешно")
```

```python
import csv
from pathlib import Path
from typing import Tuple

import librosa
import numpy as np
import torch
from torch.utils.data import Dataset


class ArmenianSpeechDataset(Dataset):
    """PyTorch Dataset для армянского речевого корпуса."""
    
    def __init__(
        self,
        data_dir: str,
        split: str = "train",
        sr: int = 16000,
        max_duration: float = 12.0,
        pad_to_max: bool = False
    ):
        """
        Args:
            data_dir: Путь к папке data_raw
            split: "train", "val" или "test"
            sr: Sample rate (16000)
            max_duration: Максимальная длительность аудио в секундах
            pad_to_max: Паддировать аудио до max_duration
        """
        self.data_dir = Path(data_dir) / split
        self.audio_dir = self.data_dir / "audio"
        self.sr = sr
        self.max_duration = max_duration
        self.max_samples = int(sr * max_duration)
        self.pad_to_max = pad_to_max
        
        # Загрузите metadata
        self.samples = []
        metadata_path = self.data_dir / "metadata.csv"
        
        with open(metadata_path, 'r', encoding='utf-8') as f:
            reader = csv.reader(f, delimiter='|')
            for row in reader:
                if len(row) >= 2:
                    filename, text = row[0], row[1]
                    audio_path = self.audio_dir / filename
                    if audio_path.exists():
                        self.samples.append({
                            'filename': filename,
                            'text': text,
                            'path': audio_path
                        })
    
    def __len__(self) -> int:
        return len(self.samples)
    
    def __getitem__(self, idx: int) -> Tuple[torch.Tensor, str]:
        """
        Returns:
            audio: Tensor формы (seq_len,)
            text: Текст транскрипции
        """
        sample = self.samples[idx]
        
        # Загрузите аудио
        audio, _ = librosa.load(sample['path'], sr=self.sr)
        
        # Обрежьте или паддируйте
        if len(audio) > self.max_samples:
            audio = audio[:self.max_samples]
        elif self.pad_to_max and len(audio) < self.max_samples:
            audio = np.pad(audio, (0, self.max_samples - len(audio)), mode='constant')
        
        # Конвертируйте в тензор
        audio = torch.from_numpy(audio).float()
        
        return audio, sample['text']


def collate_fn(batch: list) -> Tuple[torch.Tensor, list]:
    """Collate function для DataLoader с переменной длиной аудио."""
    audios, texts = zip(*batch)
    
    # Найдите максимальную длину в батче
    max_len = max(len(a) for a in audios)
    
    # Паддируйте аудио до максимальной длины в батче
    padded_audios = []
    for audio in audios:
        if len(audio) < max_len:
            padded = torch.nn.functional.pad(audio, (0, max_len - len(audio)))
        else:
            padded = audio
        padded_audios.append(padded)
    
    # Стакируйте в батч
    audio_batch = torch.stack(padded_audios)
    
    return audio_batch, list(texts)


if __name__ == "__main__":
    # Тест
```

```python
import torch
import torch.nn as nn
from typing import Optional
from transformers import AutoModel


class LoRALinear(nn.Module):
    """LoRA адаптер для Linear слоёв."""
    
    def __init__(
        self,
        in_features: int,
        out_features: int,
        rank: int = 8,
        lora_alpha: int = 16,
        lora_dropout: float = 0.1
    ):
        super().__init__()
        self.in_features = in_features
        self.out_features = out_features
        self.rank = rank
        self.lora_alpha = lora_alpha
        self.scaling = lora_alpha / rank
        
        # LoRA матрицы
        self.lora_A = nn.Linear(in_features, rank, bias=False)
        self.lora_B = nn.Linear(rank, out_features, bias=False)
        self.lora_dropout = nn.Dropout(lora_dropout)
        
        # Инициализация
        nn.init.kaiming_uniform_(self.lora_A.weight, a=5**0.5)
        nn.init.zeros_(self.lora_B.weight)
    
    def forward(self, x: torch.Tensor) -> torch.Tensor:
        """Вычислите LoRA адаптацию."""
        lora_out = self.lora_B(self.lora_dropout(self.lora_A(x)))
        return lora_out * self.scaling


class LoRAWrapper(nn.Module):
    """LoRA обёртка для MMS-TTS модели."""
    
    def __init__(
        self,
        model_id: str = "facebook/mms-tts-hye",
        rank: int = 8,
        lora_alpha: int = 16,
        lora_dropout: float = 0.1,
        target_modules: Optional[list] = None,
        freeze_base: bool = True
    ):
        """
        Args:
            model_id: HuggingFace модель ID
            rank: LoRA rank
            lora_alpha: LoRA alpha (масштабирование)
            lora_dropout: Dropout в LoRA слоях
            target_modules: Модули для LoRA (если None, используются все Linear)
            freeze_base: Заморозить базовую модель
        """
        super().__init__()
        
        self.model_id = model_id
        self.rank = rank
        self.lora_alpha = lora_alpha
        self.target_modules = target_modules or ["linear"]
        self.freeze_base = freeze_base
        
        # Загрузите базовую модель
        self.model = AutoModel.from_pretrained(model_id)
        
        # Заморозьте базовую модель
        if freeze_base:
            for param in self.model.parameters():
                param.requires_grad = False
        
        # Добавьте LoRA адаптеры
        self.lora_modules = nn.ModuleDict()
        self._add_lora_layers()
    
    def _add_lora_layers(self):
        """Добавьте LoRA слои к модели."""
        for name, module in self.model.named_modules():
            if isinstance(module, nn.Linear):
                # Пропустите слои классификации
                if "classifier" in name or "lm_head" in name:
                    continue
                
                # Создайте LoRA адаптер
                lora_module = LoRALinear(
                    in_features=module.in_features,
                    out_features=module.out_features,
                    rank=self.rank,
                    lora_alpha=self.lora_alpha
                )
                
                self.lora_modules[name] = lora_module
    
    def forward(self, input_ids: torch.Tensor, attention_mask: Optional[torch.Tensor] = None):
        """Пропустите через модель с LoRA адаптерами."""
        # Пропустите через базовую модель с перехватом
        return self._forward_with_lora(input_ids, attention_mask)
    
    def _forward_with_lora(self, input_ids: torch.Tensor, attention_mask: Optional[torch.Tensor] = None):
        """Выполните forward с LoRA адаптациями."""
        # Вспомогательная функция для добавления LoRA к выходу слоя
        hooks = []
        
        def make_hook(lora_module):
            def hook(module, input, output):
                if isinstance(output, torch.Tensor):
                    return output + lora_module(input[0])
                return output
            return hook
        
        # Зарегистрируйте hooks для LoRA слоёв
        for name, lora_module in self.lora_modules.items():
            for mod_name, module in self.model.named_modules():
                if mod_name == name and isinstance(module, nn.Linear):
                    hook = module.register_forward_hook(make_hook(lora_module))
                    hooks.append(hook)
        
        # Выполните forward базовой модели
        try:
            outputs = self.model(input_ids, attention_mask=attention_mask)
        finally:
            # Удалите hooks
            for hook in hooks:
                hook.remove()
        
        return outputs
    
    def get_trainable_params(self):
        """Получите только обучаемые параметры (LoRA)."""
        trainable_params = []
        for param in self.lora_modules.parameters():
            if param.requires_grad:
                trainable_params.append(param)
        return trainable_params
    
    def save_lora(self, path: str):
        """Сохраните только LoRA веса."""
        torch.save(self.lora_modules.state_dict(), path)
    
    def load_lora(self, path: str):
        """Загрузите LoRA веса."""
        self.lora_modules.load_state_dict(torch.load(path))


if __name__ == "__main__":
    # Тест
    print("Загружаю модель...")
    model = LoRAWrapper(model_id="facebook/mms-tts-hye", rank=8)
    
    print(f"✅ Модель загружена")
    print(f"LoRA слоёв: {len(model.lora_modules)}")
    
    # Подсчитайте параметры
    total_params = sum(p.numel() for p in model.model.parameters())
    lora_params = sum(p.numel() for p in model.lora_modules.parameters())
    
    print(f"Всего параметров: {total_params:,}")
    print(f"LoRA параметров: {lora_params:,}")
    print(f"% обучаемых: {100 * lora_params / total_params:.2f}%")
```
